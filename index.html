编程Party

  • 入门？

    • 做人嘛，开心最重要啦；先想清楚自己为什么学编程从中找到乐子就算是入门啦；至于喜欢CMU的Alice还是MIT的App Inventor或是Codecademy还是18禁的Codebabes或是Under 18的Khan Academy(然而我23依旧在Khan上挂着)更可以是蔡学镛写的《编程ING》就看个人喜好了，个人最喜欢的还是Malan机关枪开设的CS50，灰常劲爆。

  • C语言

    • 经过谨慎考虑，我们推荐一位印度朋友写的AtoZofC这个系列和Ahmed写的Art_of_Programming_Contest_SE_for_uva.pdf两本书，看得舒服了各位可以继续去找刘汝佳老师的黑书;看得吃力就去中国大学MOOC找浙大翁恺老师的C语言课和我理的C语言课吧。至于为什么要从C语言开始说起，这些书籍和课程都会跟你进行解释(其实我家门前有两棵枣树)。

    • 如果C语言勾起了您对底层的兴趣，学C还不过瘾的话，更底层的东西您可以去看王爽老师写的《汇编语言》并且登陆大叔的哈佛CS101神课主页去造计算机，该项目最小的完成人是印度13岁学生；如果需要Buff的话，可以去Coursera上两位大叔开的课，具体地址请询问果壳

  • Python

    • 目前Python主要是用C来实现的，我们权且将这看作C的延伸课，至于Python这东西到底好在哪(:3 」∠)用了就知道；记得喜欢Python一定要去Python Challenge逛逛(猹猹编者才到第六关)，还要去Eric大叔的地盘撒泡尿 (＞﹏＜)，其实我说的是去大叔开设的MIT的网络课程看看

  • Java

    • 嗯，Java这边就不多推荐了，毕竟自己也不太熟；引荐一位大仕老师，他在课程开头会问你：你看我长的帅么？(不帅)那我只有把课程讲好才对得起同学们啦( ＿ ＿)ノ｜扶墙~~~~~真心不错的课程

  • OpenCV，CUDA，Matlab。。。。甚至Ruby on Rails，PHP？

    • 编程语言就算学死也学不完，是在下输了，Matlab先去看demo同时看看范德比特大学这门课吧。好心累，只想一直用C写东西，赶什么时髦，要啥自行车╮(╯_╰)╭

软件及深度学习框架

  • Torch7: 使用Lua语言的深度学习库，被Facebook 及 Google Deepmind项目采用 [去看看]

  • Caffe: 来自伯克利的广受欢迎的深度学习框架，作者来自某省级高校某村分校咖啡制造专业 [去看看]

  • Theano: LISA实验室维护的使用Python语言的数学函数库 [去看看]

  • 一些基于Theano的深度学习库: Pylearn2, Blocks, Keras, Lasagne

  • MatConvNet: 为Matlab而生的卷积神经网络(CNN)库 [去看看]

  • Pybrain: 有了这个东西感觉自己的脑袋就可以少用一些了，几十行代码就实现了LSTM的神器 [去看看]
